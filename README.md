## üìö Installation

```
git clone https://github.com/Adivise/tt-osu-request
cd tt-osu-request
npm install
```

- After installation finishes you can use `node .` to start the bot.
- Your should to **GOLIVE** first before start the bot.

## üìÑ Configuration

- Copy or Rename `config.json.example` to `config.json` and fill out the values:

```json
{
    "username": "YOUR_TIKTOK_USERNAME",
    "bancho": {
        "username": "YOUR_USERNAME",
        "password": "YOUR_IRC_PASSWORD",
        "apiKey": "YOUR_API_KEY"
    }
}
```

- After finishes your `config.json` file should look like this:

```json
{
    "username": "respacis",
    "bancho": {
        "username": "Suntury",
        "password": "123456",
        "apiKey": "asdfghjklqwerrtyuuiop"
    }
}
```

### üìÑ Configuration Fields

- **username**: Your TikTok username

- **bancho.username**: Your osu! username

- **bancho.password**: Your osu! server password (not your account password, get it from [Get](https://osu.ppy.sh/p/api))

- **bancho.apiKey**: Your osu! api key (get it from [Get](https://osu.ppy.sh/p/api))
